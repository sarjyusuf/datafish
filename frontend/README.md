# Frontend (Node.js/React/Vite)

Retro pixel game themed UI for DataFish online store.

## Setup

```bash
cd frontend
npm install
npm run dev  # Development server
```

## Build for Production

```bash
npm run build
npm start  # Preview production build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:8080
```

## Features

- 🎮 Retro pixel game aesthetic
- 🐠 Product catalog with filtering
- 🛒 Shopping cart
- 📦 Order history
- 🔐 User authentication
- 📱 Responsive design

## Tech Stack

- React 18
- React Router v6
- Vite
- Axios
- Press Start 2P font
